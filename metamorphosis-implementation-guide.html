Mobile Speed Targets — LCP Under 2.5s on 4G

60–75% of visitors are on Android mobile with varying network quality. Every performance decision must be tested on a mid-range Android phone on 4G, not a MacBook on WiFi.

WP Rocket Configuration (Critical Settings)

  • Enable page caching — all pages cached. Set cache lifespan: 10 hours for static pages, 1 hour for home page
  • Minify CSS + JS — enabled. Test each plugin after enabling (Kadence is safe, WPForms is safe)
  • Lazy load images — enable for all <img> tags. Exclude hero image (above fold — needs to load immediately)
  • Preload hero image — add <link rel="preload"> for hero classroom image in header
  • Defer JS — defer all JavaScript except jQuery and Google Tag Manager
  • Remove unused CSS — use WP Rocket's RUCSS feature. Review on staging before enabling on live
  • Cloudflare integration — connect Cloudflare API in WP Rocket for CDN cache purge on publish

07 · Setup & Deployment Guide

Step-by-Step Launch Checklist

Phase 1 — Hosting Setup (Day 1)

  1. Purchase Hostinger Business plan — select Mumbai datacenter explicitly during signup
  2. Install WordPress via Hostinger's 1-click installer (NOT manual — 1-click sets up optimised config)
  3. Install Hostinger's free SSL certificate via hPanel → SSL → Enable
  4. Set PHP to 8.1+ in hPanel → Advanced → PHP Configuration
  5. Enable object caching (Redis) if available on the plan
  6. Connect Cloudflare — add site, update nameservers at domain registrar, wait 24h for propagation

Phase 2 — WordPress Configuration (Day 1–2)

  1. Install Kadence Theme + Kadence Blocks Pro via Appearance → Themes → Upload
  2. Create child theme (kadence-child) — copy style.css + functions.php template
  3. Activate child theme — ALL custom code goes in child theme only
  4. Set Permalinks: Settings → Permalinks → Post name (/jee-coaching-hyderabad structure)
  5. Set WordPress timezone: Settings → General → Timezone → Kolkata
  6. Install all 19 plugins (see Section 2). Activate one by one, test after each.
  7. Configure Rank Math: Run setup wizard, connect Google Search Console, configure LocalBusiness schema
  8. Install Google Tag Manager — add GTM container snippet in header/footer
  9. Create GA4 property → get Measurement ID → add to GTM
  10. Connect Microsoft Clarity → add Clarity tracking code via GTM

Phase 3 — Content Build (Day 2–5)

  1. Build all 9 pages using Kadence Blocks — follow Design Document Section 5 for section order
  2. Add faculty entries via Faculty CPT (name, subject, photo, qualifications, stats)
  3. Add student results via Results CPT (name, exam, score, year, photo with consent)
  4. Add 5+ testimonials via Testimonials CPT
  5. Configure WPForms — create enquiry form with all required fields (Section 4)
  6. Set up Zapier: WPForms → Google Sheets webhook → WATI WhatsApp → Email notification
  7. Test form end-to-end: submit test lead, verify WhatsApp fires within 60s, verify Google Sheet entry
  8. Add Google Maps embed on Contact page (exact pin: Beside SM Traders, Veg Market Road)
  9. Publish first 3 blog posts (use AI drafts, human review before publish)

Phase 4 — Pre-Launch Testing (Day 5–6)

  1. Test on 5 different Android devices: Samsung A-series (360px), mid-range (390px), large screen (428px)
  2. Run PageSpeed Insights — must hit 85+ mobile before launch
  3. Run SEO audit via Rank Math — resolve all Critical and Major issues
  4. Verify all forms submit correctly with real test data
  5. Test WhatsApp float button on mobile — confirm pre-filled message loads
  6. Test click-to-call button on mobile — confirm phone dialpad opens
  7. Check all image alt texts are filled (Rank Math flags missing alt text)
  8. Submit sitemap.xml to Google Search Console
  9. Set up UptimeRobot monitoring — alert to WhatsApp/email if site goes down

Phase 5 — Launch Day

  1. Update Google Business Profile: add website URL, verify phone, add 20+ real photos
  2. Submit site to Justdial, Sulekha, UrbanPro, Shiksha, CollegeDunia
  3. DNS cutover if moving from existing domain (or go live if new domain)
  4. Clear all caches: WP Rocket → Clear All Caches, Cloudflare → Purge Everything
  5. Final form test on live domain
  6. Announce on WhatsApp broadcast to existing student contacts

09 · Reseller Template Strategy

Build Once. Sell Many Times.

Every JEE/NEET coaching institute in India needs the same website. After building Metamorphosis, you have a proven master template. Each clone takes 6–8 hours to customise and launch.

Master Template Build Steps

  1. Build Metamorphosis as your master template on a staging subdomain (e.g., demo.youragency.in)
  2. Replace all real content with generic placeholder: "Your Institute Name", dummy faculty, placeholder results
  3. Export using WPvivid → Full Site Backup (ZIP file)
  4. For each new client: deploy fresh WordPress → restore WPvivid backup → swap content
  5. Content swap checklist: Logo, brand colors (CSS variables), faculty names/photos, results, phone number, address, Google Maps embed, WhatsApp number, Razorpay account, WATI account, GA4 property ID
